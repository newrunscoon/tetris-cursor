# Tetris 수동 QA 체크리스트

자동화 테스트(`verify-logic.mjs`) 통과 후, 브라우저에서 `index.html`을 열고 확인한다.

## 실행 환경

- [ ] `index.html` 더블클릭 또는 Live Server 없이 바로 실행됨
- [ ] 콘솔에 JavaScript 오류 없음

## 게임 플레이

- [ ] 블록이 약 0.5초 간격으로 자동 낙하
- [ ] ← / → 로 좌우 이동
- [ ] ↑ 로 시계방향 회전 (막히면 회전 취소)
- [ ] ↓ soft drop, Space hard drop
- [ ] 7종 블록 색상이 모두 구분됨

## UI

- [ ] 점수·레벨·다음 블록 미리보기 표시
- [ ] 다음 블록 미리보기가 스폰마다 갱신됨
- [ ] 줄 삭제 시 점수 즉시 반영
- [ ] 레벨 상승 시 낙하 속도 체감상 빨라짐

## 게임 오버 · 재시작

- [ ] 보드 상단까지 쌓이면 "게임 오버" 오버레이 표시
- [ ] 낙하·키 입력 중지
- [ ] "다시 시작" 클릭 시 보드·점수·레벨 초기화, 게임 재개

## 반응형

- [ ] 창 너비 축소 시 캔버스가 비율 유지하며 축소
- [ ] 모바일 너비(639px 이하)에서 사이드 패널 레이아웃 정상
