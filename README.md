# tetris-cursor

프레임워크·빌드 도구 없이 **순수 HTML/CSS/JavaScript** 만으로 만든 테트리스 게임입니다.

## 실행 방법

`index.html`을 더블클릭하거나 브라우저로 열면 바로 플레이할 수 있습니다.

## 조작법

| 키 | 동작 |
|---|---|
| ← / → | 좌·우 이동 |
| ↑ | 시계방향 회전 |
| ↓ | 빠른 낙하 (soft drop) |
| Space | 즉시 하강 (hard drop) |

## 기능

- 10×20 보드, 7종 테트로미노 (I, O, T, S, Z, J, L)
- 다음 블록 미리보기
- 줄 삭제 및 점수 (1줄 100 / 2줄 300 / 3줄 500 / 4줄 800)
- 레벨 시스템 — 점수가 오를수록 낙하 속도 증가
- 게임 오버 및 다시 시작
- 모바일 대응 반응형 레이아웃

## 파일 구성

- `index.html` — 게임 화면
- `style.css` — 스타일 및 반응형 레이아웃
- `script.js` — 게임 로직 및 렌더링
